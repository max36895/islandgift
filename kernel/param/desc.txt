Переменные для:
{
    Продукции:
        product_name - название товара
        product_description -  короткое описание товара
        product_price       - цена товара

    Статьи:
        article_name - название статьи
        article_description - описание статьи
        img_IMAGE:cen   - Вставить картинку в центре (IMAGE - название картинки с расширением)
        img_IMAGE:left  - Вставить картинку слева (IMAGE - название картинки с расширением)
        img_IMAGE:right - Вставить картинку справа (IMAGE - название картинки с расширением)
        img_index:POSITION - Вставить фоновую картинку (POSITION - позиция центр, слева, справа)
        $tab$ - поставить табуляцию (8 пробелов)
        <youtube>W#H#code<youtube>
            W - ширина вставки
            H - высота вставки
            code - код видео. Т.е. это src="https://www.youtube.com/embed/Jt3TeKUtlNk?rel=0&amp;showinfo=0" после embed/ и до ? идет код видео именно его и нужно вставлять. если вставить что-то другое, тоо видео не отобразится!


    Информация о пользователе:
        user_full_name - полное имя пользователя
        user_less_name - имя с инициалами
        user_city      - город пользователя

    Акции и спецпредложения:
        button_TEXT#$PAGE#$ - добавить кнопку (TEXT - текст внутри кнопки; PAGE - на какую страницу сайта отправляет кнопка)
        end_  - Заканчивает текст акции. Должен быть обязательно если нет кнопки, если есть кнопка его не должно быть!

}

 Обозначения:
 {
    <?php - тип файла в данном случае используется расширение как-то трогать или изменять его запрещенно!
    return - возвращает то что в '[]', опять же как-то трогать или изменять запрещенно!
    '...' => [] - название страницы, или какое-либо другое обозначение. Для чего служит прописанно в /* ... */ // П.С. '' означает что используется главная страница или в контактах используется для параметров основного сайта без поддоменов

    'title'       - заголовок страницы или название страницы
    'description' - Описание страницы
    'keyword'     - по каким словам или словосочетаниям можно найти страницы в интернете

    'section1Text' - текс из секций. Обычно это большой текст на странице. порядок секций зависит от расположения тестовых блоков
    'categoryText' - текст описание к категориям используется на странице подарки
    'text'         - текст для опросника
    'comment'      - текст для опросника, призыв к коментариям

    'item1Title' - заголовок преимуществ и особенностей они расположены в том порядке в котором на сайте.
    'item1Text'  - описание наших преимуществ и особенностей

    'address' - адрес
    'phone'   - телефон
    'mail'    - почта
    'skype'   - скайп
 }
 ! Менять обозначения запрещено !

 Разрешенные символы:
 {
    <br> - перенос на новую строку. на продуктах, статьях и акциях можно просто ставить enter, он автоматически интерлетируется как <br>

    <span class="g-color-text-3">...</span> - разукрашивает текст внутри в зеленоватый оттенок
    <span class="text-center">...</span> - размещает текст внутри по центру

    <a href='/' title='...'></a> - ссылка в href прописать путь к странице сайта без 'https://www.islandgift.ru' в title прописать короткое описание на 2-4 слова

    Не желательно писать следующее <span style="...">...</span>. Т.к. из-за этого произойдет ошибка формирования AMP страниц. Лучше попросить меня и я добавлю соответствующий класс стилей
 }
